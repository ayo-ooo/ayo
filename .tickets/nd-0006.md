# Update Provider Selection and Registration

## Objective

Update the provider selection logic to remove Docker and properly auto-detect the best available sandbox provider.

## Context

After removing Docker (nd-0002) and fixing Apple Container (nd-0003), the provider selection logic needs to be updated to:
1. Remove any Docker fallback logic
2. Auto-detect platform and available providers
3. Register built-in providers correctly

## Current State

The daemon currently hardcodes `NewNoneProvider()`:

```go
// internal/daemon/server.go:71
provider := sandbox.NewNoneProvider()
```

The provider loader registers providers but sandbox providers aren't using the registration system properly.

## Files to Modify

### internal/sandbox/register.go (NEW FILE)

Create registration for built-in sandbox providers:

```go
package sandbox

import "github.com/alexcabrera/ayo/internal/providers"

func init() {
    // Register none provider (always available)
    providers.RegisterBuiltin(providers.ProviderTypeSandbox, "none", func() providers.Provider {
        return NewNoneProvider()
    })

    // Register apple provider (macOS only)
    providers.RegisterBuiltin(providers.ProviderTypeSandbox, "apple", func() providers.Provider {
        return NewAppleProvider()
    })
}
```

### internal/daemon/server.go

Update `NewServer()` to use intelligent provider selection:

```go
func NewServer(cfg ServerConfig) (*Server, error) {
    // Select best available sandbox provider
    provider := selectSandboxProvider()
    
    // Create pool
    pool := sandbox.NewPool(cfg.PoolConfig, provider)
    // ...
}

// selectSandboxProvider returns the best available sandbox provider for this platform.
func selectSandboxProvider() providers.SandboxProvider {
    // Try platform-specific providers first
    switch runtime.GOOS {
    case "darwin":
        apple := sandbox.NewAppleProvider()
        if apple.IsAvailable() {
            debug.Log("using Apple Container sandbox provider")
            return apple
        }
        debug.Log("Apple Container not available, falling back to none")
    case "linux":
        // Future: Linux provider
        debug.Log("Linux native containers not yet implemented, using none")
    }
    
    // Fallback to none provider
    debug.Log("using none sandbox provider (no isolation)")
    return sandbox.NewNoneProvider()
}
```

```go
func (l *Loader) setActiveProviders() error {
    for typeStr, name := range l.cfg.Active {
        pt := ProviderType(typeStr)
        
        if name == "auto" && pt == ProviderTypeSandbox {
            // Auto-detect sandbox provider
            name = l.autoDetectSandboxProvider()
        }
        
        // ... rest of logic
    }
}

func (l *Loader) autoDetectSandboxProvider() string {
    switch runtime.GOOS {
    case "darwin":
        if runtime.GOARCH == "arm64" {
            // Check if Apple Container is available
            if p := sandbox.NewAppleProvider(); p.IsAvailable() {
                return "apple"
            }
        }
    case "linux":
        // Future: check for Linux container availability
    }
    return "none"
}
```
