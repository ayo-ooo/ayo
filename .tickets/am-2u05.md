# Create SandboxPlannerManager for per-sandbox instantiation

Create a manager that instantiates planners for each sandbox with proper context.

## Context
- Each sandbox (squad or @ayo) needs its own planner instances
- Manager resolves which planner to use (global default vs squad override)
- Manager provides PlannerContext with sandbox paths

## Implementation
```go
// internal/planners/manager.go

type SandboxPlannerManager struct {
    registry *Registry
    config   config.Config
    
    instances map[string]*SandboxPlanners // key: sandbox name
    mu        sync.RWMutex
}

type SandboxPlanners struct {
    NearTerm PlannerPlugin
    LongTerm PlannerPlugin
}

func (m *SandboxPlannerManager) GetPlanners(sandboxName, sandboxDir string, override *PlannersConfig) (*SandboxPlanners, error)
func (m *SandboxPlannerManager) ClosePlanners(sandboxName string) error
```

## Files to Create
- internal/planners/manager.go

## Dependencies
- am-tzub (Registry)
- am-1gwl (PlannersConfig)
- am-m5tt (Squad override parsing)

## Acceptance
- Manager instantiates planners per sandbox
- Caches instances for reuse
- Respects squad overrides
- Cleans up on sandbox close
