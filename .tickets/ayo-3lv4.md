# Multi-Agent Sandbox Collaboration

Multiple agents can work in the same sandbox. Each has own user account but shares workspace. Enables collaborative workflows where agents hand off work.

## Acceptance Criteria

- Multiple agents can join existing sandbox
- Each agent has separate user identity
- Shared /workspace mount
- ayo sandbox join <id> @agent adds agent
- ayo sandbox users <id> lists agents in sandbox
