Complete rewrite for Apple Container:

```markdown
# Sandbox Base Image

## Overview

ayo sandboxes use a minimal busybox-based image for agent execution.

## Image: busybox:stable

The base image is the official OCI busybox image, available from any
OCI-compatible registry.

## Pulling the Image

### macOS (Apple Container)

```bash
container image pull busybox:stable
```

## What's Included

- busybox with POSIX utilities
- sh shell
- Basic filesystem tools (ls, cat, grep, etc.)

## Agent Language Dependencies

Languages are installed per-agent based on config:

```json
{
  "sandbox": {
    "languages": ["go", "python", "node"]
  }
}
```

## Notes

- Images are cached locally by Apple Container
- First sandbox creation may be slow (image pull)
- Subsequent sandboxes reuse cached image
```
